# Velog Clone - Main Page

벨로그 메인페이지 클론 코딩 과제 프로젝트입니다.  
React 기반으로 벨로그의 UI 및 기본 동작을 구현합니다.

## 🛠️ 사용 기술

- React
- Vite
- Styled-Components

## 📁 폴더 구조
```
src/
├── components/ # 컴포넌트를 잘 활용해서 만들어주세용
├── pages/ # HomePage.jsx 여기 있어요!
├── assets/ # 일반적으로 이미지등을 여기 저장해서 사용합니다
└── main.jsx # 라우팅 및 전체 구조
```
